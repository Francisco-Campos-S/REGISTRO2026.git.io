Registro de Asistencia 2026

Sistema profesional para el control de asistencia por día y estudiante
📚 Instrucciones:

📥 Solo se permite importar y exportar archivos Excel (.xlsx).

🗓️ Para agregar días de asistencia, usa el botón + Agregar día.

✏️ Puedes editar los datos de los estudiantes y la asistencia por día directamente en la tabla.

📄 Para descargar la plantilla de ejemplo, usa Descargar plantilla.

💾 Para guardar los datos, simplemente edita la tabla; se guardan automáticamente.

📤 Para importar datos desde Excel, usa Importar Excel.
